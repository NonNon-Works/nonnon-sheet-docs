# csv・jsonをインポート・エクスポートする

NonNonTable は csv・ json 形式でのインポート・エクスポートをサポートしています。

* csv エクスポート: 各フィールド名をヘッダに含み、カンマ区切りでエクスポート
* csv インポート: エクスポートされる csv と同じ形式の  csv のインポートをサポート
* json エクスポート: データ部を JsonUtility の形式でエクスポート
* json インポート: エクスポートされる json と同じ形式の  json のインポートをサポート
