# データを探索する

### ソート

<img src="~/images/explore-data/sort.png" alt=""><br>

カラムヘッダーをクリックすると、そのカラムを基準に行を並び替えることができます。

クリックするたびに昇順・降順が切り替わります。

Shift+クリックでソートを解除することができます。

複数のカラムでのソートにも対応しています。

### 検索

<img src="~/images/explore-data/search.png" alt=""><br>

**Ctrl+F** で検索バーを表示できます。

- 大文字小文字を区別しない全文検索
- マッチしたセルがハイライト表示され、現在の位置と件数が表示されます（例：`1/5`）
- 次 / 前のマッチへ移動するには：
  - 検索バーの **▶** / **◀** ボタン
  - **F3** / **Shift+F3**
  - **Enter** / **Shift+Enter**（検索フォームにフォーカスがある場合）
- **Escape** で検索バーを閉じます

### フィルタ

<img src="~/images/explore-data/filter.png" alt=""><br>

カラムヘッダーにあるフィルタボタンをクリックすると、そのカラムに対するフィルタを設定できます。

フィルタのUIはカラムの型によって異なります。

| 型 | フィルタの内容 |
|---|---|
| `bool` | True のみ / False のみ を選択 |
| 数値型（`int`, `float` など） | 演算子（`=` `≠` `>=` `<=` `>` `<`）と値の組み合わせを設定。AND / OR で複数条件の組み合わせも可能 |
| `string` | 部分一致（大文字小文字を区別しない） |
| `enum` | 表示する値をチェックボックスで複数選択 |
| その他の型 | `ToString()` の結果に対する部分一致 |

複数のカラムに同時にフィルタを設定することができます。

フィルタが有効な場合、ステータスバーが表示されます。

- **↺** ボタンで、セルの値を編集した後にフィルタを再評価できます
- **Clear filters** ボタンで全フィルタを一括解除できます

フィルタが有効な間は行の追加・並び替えができません
